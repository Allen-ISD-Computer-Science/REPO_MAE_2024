Project MAE Information

Project MAE is an attempt to help tech the fundemental concepts of computing. It will use physical components to demonstrate the basics of computing. It will be based of Analytical Engine created by Charels Babbage which can be read more at the link at the bottom of the page. We will use a mixuture of convetor belts, lifts to represent the procces used for computng and tiles to represent the values used by the computer. This modern day Large scale analytical engine demonstrating visibly, clearly, and mechanically in an approachable and accessible manner the inner workings of today's computers.
